Fichiers journaux

Les fichiers journaux sont un outil très important pour diagnostiquer les problèmes tant pour les pilotes INDI que pour Ekos. Pour toute demande d'aide, il est nécessaire de joindre les fichiers journaux à cette demande afin d'aider à diagnostiquer le problème. Selon la nature du problème, vous devrez activer la fonction journal pour la fonctionnalité ou le pilote concerné. Tout activer n'est pas recommandé puisque cela générera beaucoup de données, et que la cause du problème risque de se trouver noyée dans toute cette information. Ainsi, n'activez que les fichiers journaux qui vous semblent nécessaires.
