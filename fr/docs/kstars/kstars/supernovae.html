Supernova

La page Supernova vous permet de choisir si les supernova sont affichées ou non en cochant la case Afficher les supernova. Par défaut les supernova sont dessinées comme de petit « + » de couleur orange clair. Comme pour les satellites, la couleur des supernova peut facilement être modifiée sur la page des Couleurs.

Vous pouvez régler la magnitude limite d'affichage des supernova ainsi que la magnitude limite pour les alertes. La magnitude limite est la plus faible magnitude d'un objet du ciel visible à l'œil nu ou avec un télescope.

La liste des supernova récentes peut être mise à jour dans le menu Données → Mettre à jour les données des supernova récentes.
